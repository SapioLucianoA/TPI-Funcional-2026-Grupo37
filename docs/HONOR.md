Este documento detalla el nivel de asistencia de herramientas de Inteligencia Artificial (IA) empleadas por cada integrante del equipo durante el desarrollo de este trabajo práctico. Nuestro objetivo es garantizar la transparencia, la integridad académica y la autoría intelectual del código entregado a la cátedra.
